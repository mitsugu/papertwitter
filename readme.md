# Paper Twitter Card について
## これは何
　Twitter の UI をペーパー化したジョーク html です。Web ブラウザを使ってプリンタで印刷したり、PDF 化の後に印刷して、メモパッドとして使用することができます。

## 使い方
1. Web ブラウザで index.html を表示します。
2. 直接プリントアウトするか PDF に出力してください。
3. 楽しんで。
